## 今日やったこと
Vimで複数行をまとめてコメントアウトする方法を確認した。

`Ctrl + v` で矩形選択モードに入ろうとしたが、環境によっては貼り付けとして処理されてしまった。  
そのため、代わりに `Ctrl + q` を使ったところ、矩形選択モードに入ることができた。

---

## 複数行コメントアウトの手順

```vim
Ctrl + q
```

で矩形選択モードに入る。

その後、`j` / `k` または矢印キーで対象行を選択する。

```vim
I
```

を押して、選択範囲の行頭に挿入する。

コメント記号を入力する。

```text
# 
```

最後に、

```vim
Esc
```

を押すと、選択した複数行すべてに反映される。

---

## コメント解除の手順

コメント記号を削除したい位置にカーソルを置く。

```vim
Ctrl + q
```

で矩形選択モードに入る。

削除したい `# ` の範囲を複数行分選択して、

```vim
d
```

で削除する。

## 学んだこと
### `Ctrl + v`

Vim本来の矩形選択モードに入るキー。  
ただし、Windows Terminal / PowerShell / VS Code Terminal などでは貼り付けとして処理されることがある。

### `Ctrl + q`

`Ctrl + v` が使えない環境で、矩形選択モードに入るための代替キーとして使える。

### `I`

矩形選択中に押すと、選択した複数行の同じ位置に文字を挿入できる。  
今回の場合は、各行の先頭に `# ` を入れるために使った。

### `Esc`

挿入内容を確定するキー。  
矩形選択で入力した文字は、`Esc` を押したタイミングで複数行に反映される。

### `d`

選択範囲を削除するキー。  
複数行の先頭にある `# ` をまとめて消すときに使える。

## まとめ

- 複数行コメントアウトは矩形選択を使うと便利
- `Ctrl + v` が貼り付けになる環境では `Ctrl + q` を使う
- コメントアウトは `Ctrl + q → 範囲選択 → I → # → Esc`
- コメント解除は `Ctrl + q → 範囲選択 → d`
