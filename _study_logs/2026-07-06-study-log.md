## 今日やったこと
- `tmux` の画面上でマウスホイールを使ってスクロールしようとした
- しかし、シェル側にマウスホイール操作が送られてしまい、過去コマンドが入力されるような挙動になった
- `tmux` では、デフォルト設定のままだとマウス操作が有効になっていないことを確認した
- 一時的に以下のコマンドでマウス操作を有効にした

```bash
tmux set -g mouse on
```

- マウスホイールで tmux の履歴をスクロールできるようになった
- すべての tmux セッションで有効にするため、設定ファイルに追記して恒久化した

```bash
vim ~/.tmux.conf
```

```tmux
set -g mouse on
```

- 既存の tmux セッションにも設定を反映するため、以下のコマンドを実行した

```bash
tmux source-file ~/.tmux.conf
```

## 学んだこと
- `tmux set -g mouse on` を実行すると、tmux でマウス操作を有効にできる
- `mouse on` にすると、マウスホイールで tmux のスクロール履歴をたどれるようになる
- `set -g` の `-g` は global の意味で、tmux 全体のデフォルト設定として反映する指定
- コマンドで実行した設定は一時的なもので、tmux を再起動すると消える
- 恒久的に設定したい場合は、`~/.tmux.conf` に設定を書く
- `~/.tmux.conf` を編集しただけでは、すでに起動している tmux セッションにはすぐ反映されない
- 既存セッションに反映したい場合は、`tmux source-file ~/.tmux.conf` を実行する
