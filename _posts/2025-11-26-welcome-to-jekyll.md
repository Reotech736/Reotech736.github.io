## はじめまして

Jekyll + GitHub Pagesを使って技術ブログを始めました

エンジニアとして働く中で学んだことや、個人的に試してみた技術を記録する場所として、このブログを立ち上げることにしました

## なぜJekyllを選んだか

ブログプラットフォームはたくさんありますが、Jekyllを選んだ理由は以下の通りです

### シンプルで高速

静的サイトジェネレーターなので、データベースが不要でページの読み込みが高速です
サーバー側の処理がないため、セキュリティ面でも安心できます

### GitHub Pagesとの相性

GitHub Pagesで無料ホスティングができ、Gitでバージョン管理も可能です
デプロイも自動で行われるので、記事を書くことに集中できます

### Markdownで記事を書ける

普段からMarkdownを使っているため、慣れた環境で記事を執筆できます
コードブロックやリストなども簡単に書けるので、技術記事に最適です

### カスタマイズ性

HTML/CSS/JavaScriptを使って自由にカスタマイズできます
テーマやプラグインも豊富で、必要に応じて機能を追加できます

## このブログで書いていくこと

主に以下のようなテーマで記事を書いていく予定です

### 技術学習の記録

新しい技術を学ぶ過程で得た知見や、つまずいたポイントとその解決方法を記録していきます
同じような課題に直面している方の参考になれば嬉しいです

### 開発環境の構築

実際に使っている開発環境やツールの設定方法、便利なコマンドなどを共有します
WSL、Docker、VSCodeなどの環境構築についても書いていきたいと思います

### 技術的な考察

技術トレンドや設計思想について考えたこと、実務で得た経験から学んだことをまとめていきます

### ボードゲーム

趣味で楽しんでいるボードゲームのレビューや戦略についても、たまに書くかもしれません

## 今後の予定

まずはこのブログ自体の構築過程を記事にしていく予定です
Jekyll + GitHub Pagesでのブログ立ち上げ方、カスタマイズの方法などを詳しく解説します

その他にも以下のようなトピックを扱っていきたいと考えています

- Gitの基本的な使い方と実践的なワークフロー
- AI/機械学習の基礎と実装
- Web開発におけるベストプラクティス

これから技術的な学びや気づきをこのブログで共有していきます
