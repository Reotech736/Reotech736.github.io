# はじめに

Jekyll + GitHub Pagesを使って技術ブログを始めました

## なぜJekyllを選んだか

- 静的サイトジェネレーターで高速
- GitHub Pagesとの相性が良い
- Markdownで記事を書ける
- 無料でホスティングできる

## 今後の予定

- デザインのカスタマイズ
- 記事の充実
  - まずはブログの立ち上げ方についての記事を執筆予定
  - そのほか、Gitの使い方や、 AIの話題などを扱いたいと思います

これから技術的な学びや気づきをこのブログで共有していきます
